Prepare for the tutorial in 2 steps

1. Bring a computer

If possible, bring a computer with Python installed, and with virtualenv installed.

If you don't know how to install virtualenv, please show up 15 minutes before the tutorial start time so we can help you.

2. Create an account on Slack

By the time you see this page, you should have already received an invitation to the petwitter Slack chat system. Join it!

If you haven't received one, that's OK. Send an email to: paulproteus@gmail.com.
